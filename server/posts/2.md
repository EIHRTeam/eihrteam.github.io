# 新手入门指南

欢迎来到塔卫二！这篇指南将帮助你快速上手。

## 基础操作

首先，让我们了解一下游戏的基本操作：

```javascript
// 示例代码：计算伤害
function calculateDamage(attack, defense) {
  const baseDamage = attack - defense;
  return Math.max(baseDamage, attack * 0.05);
}

const damage = calculateDamage(1000, 300);
console.log(`造成伤害: ${damage}`);
```

## 资源管理

资源管理是游戏中非常重要的一环：

1. **合理规划**：根据需求分配资源
2. **及时收集**：不要让资源溢出
3. **长期规划**：为未来的活动做准备
